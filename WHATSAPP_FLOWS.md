# WhatsApp Conversation Flows

## Flow Architecture

### State Machine Pattern
Each customer conversation maintains state in the `customers.conversation_state` JSONB field:

```json
{
  "current_flow": "onboarding|purchase|points_check|redemption|help",
  "awaiting": "phone_confirm|purchase_amount|reward_selection",
  "context": {
    "selected_reward_id": "uuid",
    "purchase_amount": 15000
  },
  "last_interaction": "2024-01-20T10:00:00Z"
}
```

  return message;
}
